# 🔢 4-bit ALU in Verilog

## 📌 Overview
This project implements a simple **4-bit Arithmetic Logic Unit (ALU)** using Verilog.  
It supports basic operations such as:
- Addition
- Subtraction
- AND
- OR
- XOR  

The design was simulated using **Icarus Verilog** and verified with **GTKWave**.

---

## 🛠 Tools Used
- **Verilog** → Hardware description language  
- **Icarus Verilog (iverilog, vvp)** → Simulation  
- **GTKWave** → Waveform visualization  
- **Git & GitHub** → Version control and portfolio hosting  
- **Ubuntu (WSL2)** → Development environment
